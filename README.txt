FutBrain - Документация на проекта
Този документ съдържа технически инструкции за конфигурация, инсталация и стартиране на платформата FutBrain.

Общ преглед
FutBrain е уеб базирано приложение за футболни фенове, което съчетава информационна база, социална интеракция и електронна търговия.

Основни компоненти:

База данни за футболни играчи и клубове.

Онлайн магазин за футболни аксесоари.

Система за публикации, коментари и дървовидна структура на отговорите.

Модерация на съдържанието чрез изкуствен интелект (Google Gemini).

Административен панел за управление на ресурсите.

Структура на проекта
Приложението е разработено като клиент-сървър архитектура (MERN стек) с TypeScript:

Backend (директория src/): Изграден с Node.js, Express и MongoDB.

Frontend (директория frontend/): Изграден с React и Vite.

Инсталиране и настройка
1. Инсталиране на зависимости
Отворете терминал и изпълнете следните команди за двата компонента:

За Backend:
cd src
npm install

За Frontend:
cd frontend
npm install

2. Конфигуриране на променливи на средата (.env)
Създайте необходимите конфигурационни файлове за правилна работа на услугите.

Backend (път: src/.env):
PORT=3000
MONGO_URI=mongodb://localhost:27017/futbrain
SECRET_KEY=vashata_taina_fraza_za_jwt
GEMINI_API_KEY=vashiyat_gemini_api_key
VITE_FRONTEND_URL=http://localhost:5173

Данни за първоначален администратор
ADMIN_USERNAME=admin
ADMIN_EMAIL=admin@futbrain.com
ADMIN_PASSWORD=Admin123!

Конфигурация за изпращане на имейли
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password

Frontend (път: frontend/.env):
VITE_API_URL=http://localhost:3000

Стартиране на приложението
Backend
Компилиране на TypeScript кода:
В папка src/ изпълнете: npx tsc

Стартиране на сървъра:
В същата папка изпълнете: npm start
Сървърът ще работи на адрес http://localhost:3000. При първи старт системата автоматично създава администраторски профил с данните от .env.

Frontend
Стартиране в режим на разработка:
В папка frontend/ изпълнете: npm run dev
Приложението ще бъде достъпно на адрес http://localhost:5173.

Функционални специфики
Съхранение на файлове
Всички качени снимки се съхраняват локално в src/uploads/, разпределени по категории:

/clubs - лога на отбори.

/players - снимки на състезатели.

/accessories - снимки на артикули.

/users - потребителски аватари.

AI Модерация
Интегрираният API ключ за Google Gemini позволява автоматична проверка на постове и коментари. Системата блокира съдържание, което бива класифицирано като неподходящо или токсично.

Контактна форма
Връзката с администрацията се осъществява чрез nodemailer, който изпраща съобщения от потребителите директно към посочения в настройките имейл адрес.

Поддръжка
Linting: Изпълнете "npm run lint" във frontend директорията.

Типизация: Проектът използва стриктна TypeScript типизация за осигуряване на стабилност при обмен на данни между клиента и сървъра.
